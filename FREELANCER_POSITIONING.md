I build Java and Spring Boot backends for SaaS products, internal tools, and MVPs. My work focuses on REST APIs, PostgreSQL data modeling, authentication, business logic, bug fixing, and backend feature delivery.

I can help you build new API endpoints, improve existing Spring Boot codebases, fix backend issues, add secure authentication, design relational schemas, and make backend systems easier to maintain as your product grows.
